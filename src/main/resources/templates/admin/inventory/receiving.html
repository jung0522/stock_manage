엑셀 템플릿 다운로드

입고 처리를 위한 엑셀 템플릿을 다운로드하세요. 테스트용 샘플 데이터가 포함된 파일로 바로 테스트해볼 수 있습니다.

빈 템플릿 다운로드 테스트용 샘플 다운로드

파일 업로드

선택한 공급업체의 상품들만 입고 처리됩니다.
.xlsx 또는 .xls 파일만 업로드 가능합니다. (최대 10MB)

사용 방법

  1. 템플릿 다운로드: 위의 "테스트용 샘플 다운로드" 버튼을 클릭하여 완성된 샘플 파일을 받으세요.
  2. 데이터 확인: 다운로드한 엑셀 파일을 열어서 데이터 형식을 확인하세요.
  3. 데이터 수정: 필요에 따라 상품명, 카테고리명, 수량, 단가를 수정하세요.
  4. 파일 업로드: 수정한 엑셀 파일을 선택하고 "입고 처리" 버튼을 클릭하세요.
  5. 결과 확인: 처리 결과를 확인하고, 오류가 있다면 해당 행을 수정 후 다시 업로드하세요.

주의사항

  • 상품명과 카테고리명은 시스템에 등록된 것과 정확히 일치해야 합니다.
  • 입고수량과 입고단가는 숫자만 입력 가능합니다.
  • 첫 번째 행은 헤더이므로 삭제하지 마세요.
  • 빈 행이 있으면 자동으로 건너뜁니다.
