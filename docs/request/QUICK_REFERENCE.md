## 📚 핵심 참조 문서

### 필수 문서 (Claude가 매번 확인해야 함)
- `docs/request/PROJECT_STATUS.md` - 현재 상태 및 다음 단계
- `docs/lectures/XX_TOPIC.md` - 해당 강의 상세 가이드

### 참고 문서
- `docs/lectures/PRODUCTION_READY_ROADMAP.md` - 전체 여정 가이드
- `docs/CLAUDE.md` - 프로젝트 기본 정보
